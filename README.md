# Media Tracker

Media Tracker is a full‑stack CRUD application for organizing and tracking personal media consumption across four categories: books, games, movies, and shows. The application is built with ASP.NET Core, Blazor Server, Entity Framework Core, and SQL Server. The project demonstrates practical use of component‑based UI design, server‑side rendering, form validation, and database‑backed state management.

The application includes a dashboard summarizing completion statistics, interactive data grids with sorting and filtering, and fully implemented Create/Read/Update/Delete workflows for each media type. It emphasizes maintainability through consistent patterns, separation of concerns, and production‑ready application structure including seeding, routing, error handling, and responsive UI layout.

## Getting started
- Clone the repository
- In Visual Studio, open Package Manager Console (View > Other Windows > Package Manager Console)
- Type the following into Package Manager Console and then press Enter:
```Package Manager Console
Update-Database
```
- After that has finished, press F5 to build and start the server and the client
