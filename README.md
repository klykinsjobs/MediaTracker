# Media Tracker

A full-stack media tracking application built with C#, ASP.NET Core Blazor, EF Core, and SQL Server. This project demonstrates full CRUD, a minimalistic dashboard, and component-based architecture. Includes support for books, games, shows, and movies.

## Getting started
- Clone the repository
- In Visual Studio, open Package Manager Console (View > Other Windows > Package Manager Console)
- Type the following into Package Manager Console and then press Enter:
```Package Manager Console
Update-Database
```
- After that has finished, press F5 to build and start the server and the client
